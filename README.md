# turbo-terminal-ios-claude-code

> Настройка Linux-сервера для продуктивной работы с Claude Code через iPhone (Termius + tmux)

---

## Быстрый старт

Одна команда — клонирует репозиторий и запускает интерактивный инсталлятор:

```bash
git clone https://github.com/mad0ps/turbo-terminal-ios-claude-code.git /tmp/turbo-setup && bash /tmp/turbo-setup/setup.sh
```

> Запускай **вне tmux**, на чистом SSH-соединении. Все промпты по умолчанию "да" — можно просто жать Enter.

---

## Что это?

Интерактивный Bash-инсталлятор, который за пару минут превращает голый сервер в удобную рабочую среду для Claude Code с айфона. Всё заточено под мобильный терминал — минимум нажатий, максимум удобства.

### Что ставится

| Компонент | Что делает |
|-----------|------------|
| **Claude Code** | Автоматическая установка через официальный скрипт (если ещё не установлен) |
| **tmux конфиг** | Prefix `Ctrl+A`, переключение окон стрелками, сплиты через `\|` и `-`, статус-бар наверху |
| **Алиасы** | `c` = claude, `cr` = resume, `cc` = continue, `q` = быстрый вопрос |
| **Навигация** | `p имя` — перейти в проект, `np имя` — новая tmux-сессия в проекте (с автокомплитом) |
| **Меню логина** | При SSH-подключении показывает активные tmux-сессии с возможностью подключиться, создать или переименовать |
| **Автосохранение** | tmux-resurrect + continuum — сессии сохраняются каждые 10 минут и восстанавливаются после перезагрузки |

Каждый компонент ставится по выбору — инсталлятор спрашивает на каждом шаге.

---

## Как работает инсталлятор

```
bash setup.sh
```

1. **Определяет систему** — ОС, пакетный менеджер (apt/dnf/yum/apk/brew), шелл (bash/zsh/fish), установленные инструменты
2. **Спрашивает что ставить** — выбираешь компоненты (по умолчанию ставится всё)
3. **Делает бэкап** — сохраняет текущие конфиги в `~/.terminal-setup-backup-<дата>/`
4. **Устанавливает** — tmux, git, Claude Code (если нужно), алиасы, конфиги, плагины
5. **Применяет** — выводит шпаргалку по горячим клавишам

Откат к исходным настройкам:
```bash
cp ~/.terminal-setup-backup-*/* ~/
source ~/.bashrc
```

## Шпаргалка

### tmux (prefix = `Ctrl+A`)

| Действие | Команда |
|----------|---------|
| Список сессий | `Ctrl+A s` |
| Новое окно | `Ctrl+A n` |
| Следующее окно | `Ctrl+A ↓` |
| Предыдущее окно | `Ctrl+A ↑` |
| Вертикальный сплит | `Ctrl+A \|` |
| Горизонтальный сплит | `Ctrl+A -` |
| Переключение панелей | `Alt + стрелки` |
| Popup окно | `Ctrl+A g` |
| Закрыть панель | `Ctrl+A x` |
| Отключиться (detach) | `Ctrl+A d` |

### Алиасы

| Команда | Описание |
|---------|----------|
| `c` | Запустить Claude Code |
| `cr` | Продолжить последнюю сессию Claude |
| `cc` | Continue с контекстом |
| `q "вопрос"` | Быстрый вопрос Claude |
| `p` | Показать список проектов |
| `p имя` | Перейти в проект (с подсказкой если не найден) |
| `np имя` | Новая tmux-сессия в проекте |
| `tl` | Список tmux-сессий |
| `tk имя` | Убить сессию |

### Меню логина

При SSH-подключении:
- **[a]** — подключиться к существующей сессии
- **[n]** — создать новую сессию
- **[r]** — переименовать сессию
- **[s]** — войти в shell без tmux
- Или просто введи имя сессии для быстрого входа

## Типичный рабочий процесс

```
1. Открыл Termius на айфоне → подключился по SSH
2. Увидел меню сессий → выбрал существующую или создал новую
3. cr → продолжил работу в Claude Code
4. Ctrl+A d → отключился, всё работает в фоне
5. Автосохранение каждые 10 минут → ничего не теряется
```

## Советы для iPhone

- **Раскладка** — переключи на английский перед `Ctrl+A`, tmux не понимает русскую
- **Свайпы в Termius** — по экрану работают как стрелки
- **Landscape** — переверни телефон горизонтально когда нужно больше ширины
- **Шрифт** — уменьши в настройках Termius для большего контента
- **Сниппеты Termius** — сохрани частые команды, запуск в одно нажатие
